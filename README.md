# Jonathan Cinnamon

Academic website and web tools for visual geography research. Associate Professor of  Geography, UBC Okanagan.

**[jonathancinnamon.com](https://www.jonathancinnamon.com)**

---

## Tools

- [Urban Change Observer](https://www.jonathancinnamon.com/urban-change-observer/) — systematic audits of street-level imagery for urban environment research
- [Image Colour Analyzer](https://www.jonathancinnamon.com/image-colour-analyzer/) — analyze colour properties across large image datasets.
- [Image Colour Visualizer](https://www.jonathancinnamon.com/image-colour-visualizer/) — visualize colour from hex/RGB/HSL/HSV values.
- [Earth Colours Mapper](https://www.jonathancinnamon.com/earth-colours-mapper/) — explore earth colours by geographic location and generate colour palettes.

All tools run entirely in the browser with no installation or server required.
